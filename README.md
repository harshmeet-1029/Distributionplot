## Distribution Plots

These plots are:

* distplot

* jointplot

* pairplot

* rugplot

## jointplot

jointplot() allows you to basically match up two distplots for bivariate data. With your choice of what **kind** parameter to compare with: 

* “scatter” 

* “reg”

* “resid”

* “kde”

* “hex”
